## بخش ۳ — صفحات جدید

### ۳.۱ `/recipes` — لیست همه دستورها
فایل: `app/recipes/page.tsx`
- Server Component (از mock-data میخونه)
- فیلتر بر اساس tag با query param: `?tag=pasta`
- نمایش grid از RecipeCard
- صفحه‌بندی ساده (آینده)

### ۳.۲ `/recipes/[slug]` — جزئیات دستور
فایل: `app/recipes/[slug]/page.tsx`
- Server Component
- عکس بزرگ با hero banner
- عنوان، توضیحات، زمان پخت، امتیاز، likes
- لیست tag ها
- اطلاعات نویسنده
- دکمه "بازگشت"

### ۳.۳ `/restaurants` — لیست همه رستوران‌ها
فایل: `app/restaurants/page.tsx`
- مشابه recipes — grid از RestaurantCard

### ۳.۴ `/restaurants/[slug]` — جزئیات رستوران
فایل: `app/restaurants/[slug]/page.tsx`
- عکس بزرگ hero
- اطلاعات: آدرس، cuisine، price level, rating, تعداد نظرات
- featured review
- لیست نظرات (review نمونه از mock)

### ۳.۵ `/categories/[slug]` — دسته‌بندی
فایل: `app/categories/[slug]/page.tsx`
- نام دسته + توضیح
- grid از RecipeCard (فیلتر بر اساس category)
- فعلاً همه recipes رو نشون میده با tag مرتبط

### ۳.۶ `/search` — نتایج جستجو
فایل: `app/search/page.tsx`
- خواندن `?q=` از search params
- فیلتر ساده از mock-data (match on title/description/tags)
- نمایش نتایج recipes + restaurants ترکیبی

### ۳.۷ `/community` — صفحه جامعه
فایل: `app/community/page.tsx`
- grid کامل از CommunityPostCard
- فعلاً همون ۳ post از mock
